Documentação da API

Guia completo para utilizar a API de Download de Mídia. A autenticação é feita via header X-API-Key.

GET/api/media

Inicia o processo de download. Responde imediatamente para tarefas longas.

Parâmetros (Query)

  • type (obrigatório): 'audio' ou 'video'.
  • name ou url (obrigatório): Termo de busca ou URL.
  • quality (opcional): Qualidade do vídeo (ex: '720p').
  • bitrate (opcional): Bitrate do áudio (ex: '192k').

Exemplo de Uso (cURL)

curl -X GET "http://localhost:25569/api/media?type=audio&name=NOME" \
-H "X-API-Key: SUA_CHAVE"

Respostas Possíveis

Resposta Rápida (200 OK)
{
  "status": "completed",
  "result": {
    "download_url": "...",
    "name": "..."
  }
}
Resposta para Tarefa Longa (202 Accepted)
{
  "status": "processing",
  "task_id": "...",
  "check_status_url": "..."
}

GET/api/tasks/<id>

Verifica o status de uma tarefa em processamento.

GET/api/download/<file>

Baixa o ficheiro finalizado. Endpoint público.

GET/api/health

Verifica a saúde da API e suas dependências.
